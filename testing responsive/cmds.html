Commands

Move cursor to see each description

?weather location

Display weather details of desired city

For example: ?weather ho chi minh


?weather -l location -u metric: Convert the temperature from Fahrenheit to Celsius

For example: ?weather houston -l houston -u metric


?air-quality location

Display air quality of desired city

For example: ?air-quality

?forecast + location

Display a chart of next 7 days weather

For example: ?forecast ho chi minh
